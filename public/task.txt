Необходимо разработать примитивный API на NodeJS + Express + Mongo/Postgre/MySQL (база на выбор).

Должно быть несколько эндпоинтов:

   1. Загрузить CSV файл. Файл необходимо распарсить и сохранить данные в базе
   2. Получить коллекцию пользователей в json формате
   3. Скачать CSV файл. Необходимо сериализовать коллекцию пользователей из базы в CSV файл и отправить его.

Структура CSV файла:
Первая строка - UserName, FirstName, LastName, Age
Остальные строки (пример) - TheBlade, Boris, Yurinov, 47
…и так далее
